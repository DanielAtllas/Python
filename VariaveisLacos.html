Variáveis e Laços de Repetição - Python

Como nas principais linguagens, é preciso declarar um nome e atribuir um valor a esta variável. Perceba que não é preciso informar o seu tipo, pois o mesmo será reconhecido na atribuição. Uma variável possui tipos de dados, podendo ela ser int (valores numéricos inteiros), float (valores numéricos quebrados), string (valores de dados em texto) e booleano (valores verdadeiros e falsos). Como no exemplo a seguir:

    var1 = Valor
OBS: Para a exibição de uma variável com algum texto, é preciso concatenar um ao outro: 
    print("texto"+var1)
  • Estruturas Condicionais

    • As condicionais, fazem com que o programa execute algo caso a condição seja verdadeira, ou seja, é usada uma comparação lógica em uma condição. A estrutura usada, é através do comando if, sendo baseado no conceito de "se...então", onde se a condição for respeitada, tal comando seguirá, além do uso do else que respeita o conceito de que caso a primeira condição não seja satisfeita, outro comando seguirá. Isso é realizado através do comando:

                if var1 operador var2:
                resultado
            else:
                resultado

    OBS: Perceba o uso de dois pontos presentes no comando, para a indicação de resultado da condicional.
  • Laçoes de repetição

    • Os laços de repetição, são condições que ao serem respeitadas, algo será feito repetidas vezes. Sendo eles:

    • while

      • Ou comando enquanto, onde o comando será repetido, enquanto a condição dos valores permanecer.

                          while var1 condição var2
                        resultado

      É possível incrementar umvalor, onde a cada repetição, o valor aumentará ou irá diminuir. Através do comando var1 = var1 + 1

    • For

      • O laço for varrerá uma lista de dados, ou seja, o comando será executado, até que o valor final seja atingido.

                          for var1 in lista
                        resultado

      Ou, é possível usar um range, onde o looping terminará ao chegar no valor final:

                          for var1 in range(VF):
                        resultado

      Ou até mesmo manipular este range, declarando um valor inicial, um valor final e a sequência contada:

                          for var1 in range(VI,VF,SEQ):
                        resultado
  • Strings

    • Strings são variáveis que recebem valores do tipo texto, os quais são representadas por aspas duplas. Ou seja, todo tipo de dado que for considerado como texto (varchar na maioria dos casos) é chamado de string. Seguem alguns comandos manipuláveis de strings:

    • Tamanho da string

      • É possível exibir o tamanho da string, sendo exibido a quantidade de caracteres presentes no mesmo. Para isso, é usado o comando len, como no exemplo seguinte:

                          len(var1)
    • Parte da string

      • Através do comando [:] é possível exibir apenas parte da string, onde deve ser informado o caracter inicial e o final (posição), dessa forma:

                          print(var1[PI:PF]
    • Alteração da fonte

      • Um dos métodos disponíveis presentes em strings, é a alteração da fonte da string maiúsculo ou minúsculo. Através do comando lower/upper:

                          var1.upper()
    • Converção da string

      • Pelo comando split() é possível converter a string para uma lista, de forma que é informado um parâmetro, e a partir dele, todo o resto será parte da lista.

                          var1.split("parametro")
    • Procura

      • Também é possível encontrar a posição que parte de uma string é encontrada, devendo ser informado qual parte deseja.

                          var1.find("parametro")
    • Reconstrução

      • O último método para strings, é a reconstrução de parte da string por uman nova. Porém é preciso informar qual dado deverá ser reconstruído.

                          var1.replace("dado","novo dado")
