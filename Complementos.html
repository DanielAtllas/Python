Complementos - Python

  • Objetos

    • O python é uma linguagem orientada a objetos e por isso, "tudo" nele são objetos, sendo estes objetos chamados também de instâncias de classes. Como por exemplo ao criar e declarar uma variável, o python a transforma em um objeto. Além disso, é importante a compreenção de que esses objetos possuem atributos e métodos.

                Métodos são ações que podem ser aplicadas:
                
                    objeto.método()
                    
            Atributos são características:     
                
                    objeto.atributo
  • Funções

    • As funções, nada mais são do que scripts a serem executados separadamente do principal, de forma a não interferir de maneira direta no script principal a menos que seja chamado. É necessário o comando def seguido pelo nome da função e seus parâmetros, onde ao ser chamado no script principal, devem-se passar os valores a serem usados.

    OBS: Lembre-se que as variáveis de u função só são válidas enquanto função. Assim como diferente de outras linguagens, as funções devem ser encontradas na parte superior do script, em cima do código principal. 
                def nome (valores)
                return script
  • Arquivos

    • Através do python, é possível manipular arquivos de formas a serem usados pelo código escrito. Primeiramente, é necessário abrir o arquivo (recomenda-se atribuir o comando a uma variável)

                var1 = open(arquivo,modo)

    Perceba que é necessário informar o modo do arquivo, o qual indicará qual manipulação será feita. Os modos existentes são:

    1. r = Somente leitura

    2. w = Somente Escrita (Excluimento do dado existente)

    3. a = leitura e escrita

    Além disso, estão presentes também os comandos de leitura, sendo eles:

    1. read() = Leitura do arquivo

    2. readline() = Leitura de linha do arquivo

    3. readlines() = Leitura e armazenamento em uma lista

    Ou seja, há primeiramente 2 opções de criação de arquivo pelo próprio script ou por outro programa:

                var1 = open("arquivo","w")

    Depois um variável para leitura do arquivo

                var2 = arquivo.read()

    Agora troque o modo do arquivo para a e insira dados no mesmo.

                var1.write("registro")

    Por fim, não se esqueça de fechar o arquivo (possíveis problemas podem ser gerados sem tal função).

                var1.close()
  • Listas

    • As listas, funcionam exatamente como os ponteiros, onde dados serão inseridos, ou excluídos, da lista e manipulados dentro dela. O primeiro passo a ser feito, é criar a lista (podendo ela já conter ou não dados).

                var1 = []

    A partir de então, são seguidos os comandos da manipulação, sendo eles:

    • append() = Inserção de registros
    • len() = Quantidade de registros
    • del() = Remoção de registros
                lista.append(dado)

    Além dos comandos for e if, podendo eles:

    • Percorrer toda a lista
      •                     for var1 in lista
                        print var1
    • Extrair registro pertencente
      •                     for var1 in lista
                        print("dado presente")

    Além dos comandos de manipulação da lista, é possível exibir o seu conteúdo de forma ordenada, podendo ser ordenação crescente, decrescente, reversa ou não.

    • lista.sort()

      • Ordenação crescente (Independente da posição)

    • lista.sort(reverse=true

      • Ordenação decrescemte (Independete da posição)

    • lista.reverse()

      • Ordenação reversa

    • var1 = sorted(lista)

      • Exibição padrão da lista
