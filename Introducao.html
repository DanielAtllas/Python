Introdução - Python

O python é uma das mais populares lingaguens de programação usada nas mais diveras empresas, criada no ano de 1991, sendo usada em serviços web, desenvolvimento de softwares e sistemas, casos mamemátics e etc. Além disso, devido ao seu poder de criação de scripts, o python tem capacidade de rodar em diferentes plataformas e sistemas, seja ele Windows, Linux ou Max, estando escrito inteiramente em inglês.

Para que o scripts sejam criados na linguagem python, é necessário que o mesmo se encontre instalado em sua máquina (alguns computadores possuem a chance de já possuirem o client instalado, por isso, basta um comando de python --version dado na linha de comando para verificação). Para isso, é necessário baixar a versão do Python 3 na máquina usada, podendo ser encontrado no link: https://www.python.org/. Além disso, é recomendado baixo o sublime text, como um compilador para a escrita da linguagem, ou seja, nele será possível escrever e visualizar todo o script e seu andamento, fazendo com que o python 3 apenas o compile e execute na máquina em questão.

É importante que ao se salvar qualquer script na linguagem python, o arquivo seja salvado com o ".py" (que tornará o executável como um python propriamente dito). Perceba que diante disso, o script pode ser escrito em qualquer plataforma de editor de texto, como o próprio NotePad por exemplo, desde que o arquivo seja salvo diretemante como ".py" (porém, recomendado que se use um compilador próprio para isso).

  • Comentários

    • Outra definição que pode ser de tamanha importância durante o entendimento de um script, são os comentários que o compõem. Esses comentários são dados de 2 formas:

    1. # - Que tornará toda uma linha como comentário, ou seja, qualquer escrita dentro desta linha, o leitor a entenderá como um comentário e a ignorará.

    2. """ - Que tornará N linhas como comentários, devendo partir do programador a repetição do comando para indicar o fim da caixa de comentários.

  • Python Syntax

    • A sintaxe do python, é o que fará a exibição de algum texto para o usuário, seja ele um texto propriamente dito, uma variável ou a soma dos dois. Esse comando de exibição é semelhante ao da linguagem C, onde é dado um print(), com o texto dentro da função. como no exemplo a seguir:

                print("Hello World!");

    Que fará com que o texto Hello World! seja mostrado em tela. Além disso, esta função possui algumas características de mostragem, como por exemplo a exibição do valor de uma variável ou de um texto concatenado a uma variável, por exemplo:

                print(var);
            
            ou
            
            print("Texto ",var);

    Assim como, também é possível pular linhas, sejam elas quantas quiser, dado pelo comando de \n dentro da função print(). Logo, se utilizar de N \n na função (seja antes ou após o texto), serão puladas as N linhas, como no exemplo abaixo:

                print(\n\n texto \n\n);
  • Variáveis

    • O python, diferente de outras linguagens, possui uma peculiaridade sobre a sua definição de variáveis, uma vez que não é preciso informar o tipo da variável, pois em várias outras linguagens, ao se declarar uma variável, é necessário passar como informação, o seu tipo, seja ele int, float ou string. Porém, no python, a própria linguagem define esse tipo da variável ao informar o valor da variável, ou seja, se em uma variável X for dado um valor numérico inteiro, a própria linguagem subentende a variável como do tipo int. Logo, sua declaração se dá por o nome da variável e seu valor, como no exemplo abaixo:

                var1 = 25;       (int)
            
            var2 = 2.5;      (float)
            
            var3 = "Hello";  (String)

    Ou seja, as variáveis funcionarão como conteiners para o armazenamento de valores de dados. Além disso, devido ao fato de não haver uma declaração do tipo da variável, em qualquer momento do programa, o usuário poderá inserir nela um valor completamente diferente de seu atual. Imagine que uma variável X é criada e é atribuido a ela o valor 5 (inteiro), posteriormente, há a possibilidade de inserir um valor de string como "Marcos", por exemplo (Além de que, é importante lembrar, que os valores do tipo string são por sua vez, case-sensitive).

    Como nas demais linguagens, uma variável não pode possuir caracteres especiais e deve respeitar as regras de criação, como por exemplo, o fato de não poder haver espaço entre as letras de seu nome ou iniciar com um valor numérico.

    Outro conceito importante sobre as variáveis, é que toda variável criada durante o programa principal, é tida como uma variável global, ou seja, será possível usa-la durante o programa e até mesmo dentro de funções (sem necessidade de cria-la de novo). Porém, o Python disponibiliza da possibilidade de variáveis globais dentro de funções, as quais podem ser criadas enquanto função e usadas da mesma forma enquanto programa principal. Essas dadas pelo prefixo:

                global var;

    Por sua vez, essas são chamadas de global keywords.

    Por mais que as variáveis não necessitam do prefixo indicando o tipo de cada variável, a leitura/atribuição de um valor a ela, torna-a de um tipo especifico. Diante disso, os tipos possíveis das variáveis se constituem em:

    • Text Type: str

    • Numeric Types: int, float, complex

    • Sequence Types: list, tuple, range

    • Mapping Type: dict

    • Set Types: set, frozenset

    • Boolean Type: bool

    • Binary Types: bytes, bytearray, memoryview

    Atrvés do comando de exibição (print()) integrado com a função de type(), é possível exibir e assim identificar qual o tipo atual de uma determinada variável, dado isso pelo comando:

                print(type(var));
