Introdução - Python

Assim como as demais linguagens, a sintaze dos scripts é a mesma, porém o python possui algumas diferenças, e uma delas é a instalção do programa. Pois é necessário o uso de 2 programas, o próprio python (como um compilador dos scripts), sendo o mais utilizado python3, e o sublime text3, que servirá de interface de script. É importante que todos os scripts estejam salvos em .py, dessa forma, o python o entenderá como programa executável.

OBS: É usado o comando ctrl + B para a compilação e execução do script.

É recomendado que no início do script, coloque-se um chamador de caractres especiais, o qual irá torna-los válidos durante todo o programa, dado através do comando :

    #-*- coding: utf-8 -*-

Falando nisso, para tornar algo um comentário, sendo muito semelhante ao shell, usa-se do símbolo # para tornar toda a linha um comentário, ou então, usa-se três aspas duplas para tornar várias linhas como comentário. Além disso, outro comando que será extremamente útil no decorrer de um programa, é a capacidade de limpar tela, o qual é preciso inserir o comando inteiro, toda vez que deseja ser realizado, importando a biblioteca "os" e executando-a. Dado pelo seguinte comando:

    import os
    os.system('cls')
